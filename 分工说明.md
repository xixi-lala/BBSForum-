# BBS技术社区 - 项目分工说明

## 项目概述

BBS技术社区是一个基于 Jakarta Servlet 5.0 + JSP + JSTL 的论坛系统，使用嵌入式 Tomcat 10.1 启动，连接 MySQL 8.0 数据库。

---

## 全组协作文件

| 文件 | 说明 | 负责人 |
|------|------|--------|
| `database/init.sql` | 所有人把自己设计的表交给组长，由组长合并 | 组长汇总 |
| `layouts/main.jsp` | 公共布局，组员新增页面时通知组长统一修改 | 组长维护 |
| `WEB-INF/web.xml` | 公共配置 | 组长维护 |
| `测试文档.md` | 各自写自己的模块测试用例，组长汇总 | 全员+组长汇总 |

---

## 分工原则

1. **每人负责的模块自始至终独立完成** — 从数据库设计 → 后端代码 → 前端页面，全流程负责
2. **互不依赖** — 组员A不需要等组员B的代码就能做自己的功能
3. **接口清晰** — 公共的布局和工具类由组长统一提供，组员直接使用
4. **测试自测** — 每人对自己负责的功能写测试用例
